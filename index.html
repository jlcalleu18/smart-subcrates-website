🚀 Quick Start Guide

🚀 Guía Rápida

  • Download the app
  • Unzip the file
  • Run the app (double click)
  • Select your Music and Subcrate folders
  • Click "Create Subcrates"
  • Descarga la aplicación
  • Descomprime el archivo
  • Ejecuta la aplicación (doble clic)
  • Selecciona tus carpetas de Música y Subcrates
  • Haz clic en "Crear Subcrates"

❓ Frequently Asked Questions

❓ Preguntas Frecuentes

  • Is this safe? – Yes, it runs locally and doesn't upload your music.
  • Does it move files? – No, it only generates .crate files.
  • Supported OS? – Windows & macOS
  • Do I need Serato? – You need Serato installed to use the crates.
  • ¿Es seguro? – Sí, funciona localmente y no sube tu música.
  • ¿Mueve los archivos? – No, solo genera archivos .crate.
  • Sistemas operativos compatibles – Windows y macOS
  • ¿Necesito tener Serato? – Sí, debes tener Serato instalado para usar los subcrates.
